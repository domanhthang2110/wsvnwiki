# Codebase Overview

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **State Management**: Zustand
- **Database/Auth**: Supabase
- **Editor**: TipTap
- **Icons**: Lucide React

## Key Concepts
- **Server Components**: Used by default in `app` directory.
- **Client Components**: Marked with `'use client'` for interactive UI.
- **Image Optimization**: Using `wsrv.nl` via `imageLoader` for external Supabase images.
